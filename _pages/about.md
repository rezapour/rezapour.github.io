Hey, I'm Reza 👋
I'm a software engineer with a focus on Android development. I genuinely enjoy the craft — good software architecture, clean abstractions, and writing code that's easy to reason about. I don't believe in one-size-fits-all solutions; different problems call for different architectural approaches, and finding the right one for the right context is something I find genuinely interesting.

I try to keep up with the latest tools and approaches in Android, though I'm realistic enough to know that's not always possible in practice. The pursuit of it is half the fun.
I care a lot about open source and try to keep side projects going — partly to learn, partly to give something back to the community that's taught me so much.
Outside of code, I'm into movies, video games, running, hiking, cooking, and reading. A fairly busy list, but it keeps life interesting.

This space is where I'll be sharing things as I go — code snippets, architecture thoughts, things I'm learning, and probably a few non-technical posts along the way too. I'm just getting started here, so check back soon.
